# BKies - Smart Campus Mobility 🚲

**BKies** is a requirement-driven bike-sharing system designed specifically for **Ho Chi Minh City University of Technology (HCMUT)** students and faculty.

## 🚀 Key Features
- **RFID Integration:** Unlock bikes instantly using your existing Student ID.
- **Pay-per-distance:** Fair billing system based on localized wheel sensors.
- **Eco-friendly:** Promoting green transportation within campus boundaries.

## 🛠 Tech Stack
- **Frontend:** React + Vite + Tailwind CSS
- **Animations:** Framer Motion
- **Hardware Simulation:** C++, RFID RC522, Hall Effect Sensors (Conceptual)

## 📋 Engineering Specifications
- **Unlock Speed:** ≤ 3 Seconds
- **Bike Weight:** 18.0 kg
- **Frame Strength:** > 400 MPa
- **Water Resistance:** IPX4
